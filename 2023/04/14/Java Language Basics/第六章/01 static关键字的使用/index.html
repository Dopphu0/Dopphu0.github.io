01 static关键字的使用

2023-04-14 Java Language Basics 第六章 面向对象(下) 641 字 
/**
 * static关键字的使用
 * 
 * 1.static：静态的
 * 2.static可以用来修饰：属性、方法、代码块、内部类
 * 
 * 3.使用static修饰属性:静态变量（或类变量）
 * 		3.1属性按是否用static修饰分为：非静态变量和静态变量（实例变量）
 * 		实例变量：我们创建了类的多个变量，每个对象都独立的拥有一套类中的非静态属性，当修改其中一个对象的
 * 			非静态属性时，不会导致其他对象中同样的属性值的修改
 * 		3.2静态变量：所有类的对象共享同一个静态变量。当通过某一个对象来修改其值时，
 * 			其他对象调用该静态变量的时候，是修改过的
 * 		3.3 static修饰属性的其他说明：
 * 			1）静态变量随着类的加载而加载
 * 			
 * 			2）静态变量的加载要早于对象的创建
 * 
 * 			3）由于类只会加载一次，则静态变量在内存中也只会存在一份，存在方法区的静态域中
 * 
 * 			4）		类变量		实例变量
 * 			类		yes			no
 * 			对象		yes			yes
 * 
 * 4.使用static修饰方法：静态方法
 * 			me：主要原因是生命周期不同
 * 			1）随着类的加载而加载，可以通过“类.静态方法”的方式进行调用
 * 			2）			静态方法		非静态方法
 * 				类		yes			no
 * 				对象		yes			yes
 * 			3）静态方法中只能调用静态方法或属性
 * 			4）非静态方法中，既可以调用非静态方法或属性，也可以调用静态的方法或属性
 * 5.static注意点
 * 		在静态的方法内，不能用this、super（me：静态属性或方法前省略的不是this，而是类）
 * 		要从生命周期的角度进行理解		
 * 6.如何确定一个属性是否要声明为static的？
 * 		> 属性是可以被多个对象共享的，不会随着对象的不同而不同\
 * 		> final修饰的变量就不再是变量了，而是常量了，常量习惯上也常常声明为static
 * 		
 * 	 如何确定一个方法是否要声明为static的？
 * 		> 操作静态属性的方法通常就设置为静态的
 * 		> 工具类中的方法，习惯上声明为static的。比如：Math、Arrays、Collection
 * 		
 */		
package chapter06;

/**
 * @author Dopphu
 *
 */
public class Test01 {
	public static void main(String[] args) {
		
		Chinese.nation = "中国";
	
		System.out.println(Chinese.nation);
		
		Chinese c1 = new Chinese();
		c1.nation = "CHN";
		Chinese c2 = new Chinese();
		
		System.out.println(c2.nation);
		
	}
}
//中国人
class Chinese{
	String name;
	static String nation;
}
