《中华人民共和国刑法》第二百八十六条

IP地址详解

计算机、交换器、路由器等网络设备用电缆等传输介质进行连接——>网络

世界互联——>互联网

适配器——>网卡

MAC地址理论上不能改，实际上是可以的，比如ARP攻击、中间人攻击（流量都经过我）

MAC地址——>六段、十六进制

IP由两部分组成

  • 网络部分（NETWORK）
  • 主机部分（HOST）

我们能接触到的IP只有A（1-126）、B（128）、C（192）

A B C
网络位 1（8位） 2（16位） 3（24位）
主机位 3 2 1

127为本机回环地址，用来测试本机

子网掩码口诀：网络位全为1，主机位全为0（但是并不符合所有情况）

结论一： 已知网络位和主机位可得

1）子网掩码（网络位全为1、主机位全为0）

2）求网段：网络位不变，主机位为0

​ e.g. IP 192.168.1.1；子网掩码 255.255.255.0； 网段 192.168.1.0（1.255【广播地址】和1.0【代表整个网段】都不能用）【me】

另一种表示方法：192.168.1.1/24

私有地址

  • 满足企业用户在内部网络中使用的需求
  • 私有地址不能在互联网上使用
  • 私有地址包括三组
    • A类：10.0.0.0~10.255.255.255
    • B类：172.16.0.0~172.31.255.255
    • C类：192.168.0.0~192.168.255.255

DHCP-Server自动分配（自己搭建，自己规定如何分配）

  • IP（没有DHCP-Server，则电脑自己给自己一个168开头的）
  • 网关（要和IP地址要在同一段）
  • DNS（8.8.8.8——谷歌的国际DNS）
  • 子网掩码
